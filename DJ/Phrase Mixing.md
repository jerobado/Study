# Phrase Mixing

| Pattern            | Description                                           | Best Use                                   |
|--------------------|-------------------------------------------------------|--------------------------------------------|
| Outro → Intro      | Fade out the ending section of Track A while bringing in the beginning of Track B. | Classic, safe transition; works well for most genres. |
| Outro → Breakdown  | End of Track A overlaps with a breakdown (low-energy section) of Track B. | Smooth energy reset before building up again. |
| Breakdown → Intro  | Track A’s breakdown overlaps with Track B’s intro.    | Creates anticipation; good for building tension. |
| Drop → Drop        | Align the high-energy drops of both tracks.           | High-impact, crowd-pleasing transition; best when keys/energy match. |
| Verse → Verse      | Vocals from Track A lead into vocals from Track B.    | Tricky—avoid vocal clashes; works if one track has sparse vocals. |
| Build-up → Drop    | Track A’s build-up leads directly into Track B’s drop.| Dramatic, powerful transition; great for hype moments. |
| Breakdown → Breakdown | Both tracks overlap during their low-energy sections. | Smooth, subtle transition; useful for genre shifts. |
| Chorus → Chorus    | Main hook of Track A flows into main hook of Track B. | Keeps energy consistent; works well in pop/EDM. |
