# WinGet
- WinGet is a command line tool in for installing and maintaining applications in Windows.
- Is the client interface to the Windows Package Manager service.

## Usage

### Searching an app
```
winget search <appname>
```

### List app installed in your Windows
```
winget list
```

### List app by query
```
winget list cmake
```

### Upgrade an app
```
winget upgrade cmake
```
