- new `TimeProvider`` and `ITimer` interface add _time abstraction_ functionality

- `System.Security.Cryptography.RandomNumberGenerator.GetItems`

**Shuffle&lt;T>()**

-  `RandomNumberGenerator.Shuffle<T>(Span<T>)`
